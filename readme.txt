Simplish is a WordPress variant of the Scribbish theme for blogs.

== Description ==
Simplish imitates the visual style of [Jeffrey Hardy][1]'s Scribbish (c.2005).
It began in 2008 as a patchset for [Kenny Pitt's ScribbishWP][2]. Simplish
implements the hAtom microformat specification for posts and hCard for post
authors.
